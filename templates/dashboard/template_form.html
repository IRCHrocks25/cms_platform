How to annotate

Wrap each editable block in a data-section. Mark editable elements with data-edit="section.field" and a data-type.

type binds to
textelement text
richtextinner HTML (contenteditable)
imagesrc attribute
colorinline background-color
linkhref attribute

Minimal example:

<section data-section="hero" data-label="Hero" data-group="Home">
  <h1 data-edit="hero.title" data-type="text" data-label="Headline">Welcome</h1>
  <img data-edit="hero.image" data-type="image" data-label="Photo" src="...">
</section>

CSS variables inside <style data-tokens> auto-generate brand color pickers (e.g. --primary).
